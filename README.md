# DSA Pattern-Based Learning Journey

This repository documents my pattern-based DSA learning journey.

After completing a 60-day GeeksforGeeks streak, I decided to shift my focus from solving random questions to understanding the underlying patterns used in problem solving.

## Goal

* Learn DSA pattern by pattern
* Build strong problem-solving intuition
* Understand the "why" behind solutions
* Maintain consistency through daily practice
* Document progress publicly

## Patterns Covered

### Queue Patterns

#### Basic Queue Simulation

* Number of Recent Calls
* Time Needed to Buy Tickets
* Implement Queue using Stacks
* Implement Stack using Queue
* Design Circular Queue

#### Sliding Window + Deque

* First Negative Integer in Every Window of Size K
* First Non-Repeating Character in a Stream
* Longest Continuous Subarray With Absolute Diff ≤ Limit
* Sliding Window Maximum

#### Monotonic Deque

* Continuous Subarrays
* Count Subarrays With Cost Less Than or Equal to K
* Shortest Subarray with Sum at Least K
* Max Value of Equation
